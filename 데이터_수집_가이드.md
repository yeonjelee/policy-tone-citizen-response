### 2-2. API 방식 (자동화 원할 경우)

- 네이버 데이터랩 검색어 트렌드 API 공식 지원
- API 콘솔: https://developers.naver.com/apps/#/register
- 엔드포인트: `POST https://openapi.naver.com/v1/datalab/search`
- 요청 파라미터: `startDate`, `endDate`, `timeUnit: "month"`, `keywordGroups`
- 인증: Client ID / Client Secret (무료 앱 등록 후 즉시 발급)

```python
import requests, json

headers = {
    "X-Naver-Client-Id": "YOUR_ID",
    "X-Naver-Client-Secret": "YOUR_SECRET",
    "Content-Type": "application/json"
}
body = {
    "startDate": "2010-01-01",
    "endDate": "2024-12-31",
    "timeUnit": "month",
    "keywordGroups": [{"groupName": "예금금리", "keywords": ["예금 금리"]}]
}
r = requests.post("https://openapi.naver.com/v1/datalab/search",
                  headers=headers, data=json.dumps(body))
```

> ⚠️ 네이버 데이터랩은 2016년 이전 데이터가 제한적. 2010~2015 구간은 수동 확인 필요.

### 3-1. 한국은행 ECOS (경제통계시스템)

| 변수 | 통계표명 | 통계표 코드 | 주기 |
|------|---------|------------|------|
| `base_rate` | 기준금리 | `722Y001` | 일별 → 분기말 |
| `cpi_yoy` | 소비자물가지수 전년동월비 | `901Y009` | 월별 |
| `exchange_rate` | 원/달러 환율 (매매기준율) | `731Y001` | 일별 |
| `kospi_return` | KOSPI 지수 | `802Y001` | 일별 |

**수집 방법 — ECOS Open API (권장):**

- 서비스 신청: https://ecos.bok.or.kr/api/#/DevGuide/APIKeyApply
- 무료 발급, 즉시 사용 가능
- 엔드포인트 예시:

```
https://ecos.bok.or.kr/api/StatisticSearch/[API_KEY]/json/kr/1/100/722Y001/Q/2010Q1/2024Q4
```

```python
import requests, pandas as pd

API_KEY = "YOUR_ECOS_KEY"
def ecos_fetch(stat_code, freq, start, end):
    url = (f"https://ecos.bok.or.kr/api/StatisticSearch/{API_KEY}"
           f"/json/kr/1/10000/{stat_code}/{freq}/{start}/{end}")
    data = requests.get(url).json()
    rows = data["StatisticSearch"]["row"]
    return pd.DataFrame(rows)

df_rate = ecos_fetch("722Y001", "Q", "2010Q1", "2024Q4")
df_cpi  = ecos_fetch("901Y009", "M", "201001", "202412")
```

**수동 다운로드 대안:** https://ecos.bok.or.kr → 통계검색 → 해당 통계표 선택 → 엑셀 다운로드
